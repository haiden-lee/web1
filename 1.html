HTML

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications.

HTML is the dominant markup language for web pages, which means hypertext markup language (HyperText Markup Language, cultural language: hypertext markup language, hyperbon markup language). HTML provides a way to create a structural document from links, quotes and other items, as well as representing architectural meaning for text such as titles, paragraphs, lists, etc.
